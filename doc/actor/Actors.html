Actors

Any in-game object.

Structures

Actor

Actor: {
    wire,
    unwire
}
wire function(self) called to wire up the actor's components
unwire function(self) called to wire up the actor's components

ProtoActor

ProtoActor: {
    type,
    x,
    y,
    arg
}
type string used by actor factory to determine which creation function to call
x number x position
y number y position
arg object initialization arg for actor
structure varies with type of actor

Functions

create function(protoActor) creates an actor based on protoActor
could return undefined
not virtual function, but must be implemented and exported by all types of actor
