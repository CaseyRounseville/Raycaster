Animation

Changing image over time. Can be applied to billboards, overlays, and blocks.

Structures

Animation

Animation: {
    frames
}
frames [] the frames of the animation
unregisterBillboard function(self, billboard) unregisters a billboard
registerOverlay function(self, overlay) registers an overlay
unregisterOverlay function(self, overlay) unregisters an overlay
render function(self) render everything
first, the map and billboards are drawn, then the overlays

AnimationFrame

AnimationFrame: {
    dur
}
dur number duration of the animation frame, in game frames
unregisterBillboard function(self, billboard) unregisters a billboard
registerOverlay function(self, overlay) registers an overlay
unregisterOverlay function(self, overlay) unregisters an overlay
render function(self) render everything
first, the map and billboards are drawn, then the overlays

Animator

Animator: {
    anims,
    animNum,
    frameNum,
    delayCounter,
    cbs,
    playbackMode
}
anims [] registers a billboard
animNum number unregisters a billboard
frameNum number registers an overlay
delayCounter number unregisters an overlay
cbs [] render everything
first, the map and billboards are drawn, then the overlays
playbackMode number render everything
first, the map and billboards are drawn, then the overlays

File Structure

All animations stored in a file are required to have a version field.
The version field is used to determine which loading strat to use to load the animation.
