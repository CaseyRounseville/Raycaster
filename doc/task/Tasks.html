Tasks

Hold items and returns a handle upon adding one.
Keepers are fixed sized containers that allow their items to be accessed with a handle, which is just an integer.

Structures

Task

Task: {
    cbs,
    tick
}
cbs [] task completion callbacks
callbacks are called once, in the order they were registered, with zero arguments, upon task completion
tick function(self) called each frame until the task is completed
returns true if completed, false if not

Functions in Task module

create function() creates a task
registerCallback function(self, cb) registers a task completion callback
unregisterCallback function(self, cb) unregisters a task completion callback

Functions in TaskSystem module

registerTask function(task) registers a task
unregisterTask function(task) unregisters a task
