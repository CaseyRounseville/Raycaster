Keeper

Hold items and returns a handle upon adding one.
Keepers are fixed sized containers that allow their items to be accessed with a handle, which is just an integer.

Structures

Keeper

Keeper: {
    max,
    objs
}
max number the max value of a handle
handles start at zero and are reused
keepers are fixed size containers
objs [] holds the objects

Functions

create function(max) creates a keeper with the max value specified
add function(self, obj) add an object to the keeper
returns a handle for the object
remove function(self, handle) removes the object associated with the specified handle
returns the removed object
