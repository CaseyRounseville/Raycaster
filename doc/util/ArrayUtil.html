ArrayUtil

Array utility functions for convenience.

Functions

add function(arr, i, val) adds val to arr at index i
remove function(arr, val) removes val from arr
setRangeConst function(arr, start, end, val) sets arr[start] to arr[end - 1] to val
setRangeFn function(arr, start, end, fn) sets arr[start] to arr[end - 1] to the return value of fn(i),
where i is a counter in a loop from start to end - 1,
and fn is called each iteration
